# PrivacyGuard AI Powered Data Security System

## Description
PrivacyGuard is a simple web application that analyzes application permissions and identifies possible privacy risks. It helps users understand how their data may be accessed and provides warnings for sensitive permissions.

## Features
- Permission-based risk analysis
- Risk score calculation
- User-friendly interface
- Instant result display

## Technologies Used
- Java
- HTML
- JavaScript

## Objective
To improve user awareness about data privacy and reduce misuse of sensitive permissions.

## How to Use
1. Enter the application name
2. Select the permissions
3. Click "Check Risk"
4. View the result
